1. Завдання Переход на сторінку

На сторінці є дві кнопки. При натисканні на першу кнопку користувач повинен ввести в prompt посилання, при натисканні на другу – переадресовується на інший сайт (за раніше введеним посиланням).

2. Завдання. Використання подій

Створіть HTML-сторінку з декількома кнопками. Ваше завдання - створити обробник подій для батьківського елементу, який відслідковуватиме кліки на всіх кнопках. Покроковий план: Створіть HTML-елементи: батьківський контейнер з декількома вкладеними кнопками. Додайте обробник подій до батьківського контейнера, який відслідковуватиме кліки. Після кліку на будь-якій кнопці, виведіть повідомлення про те, яка саме кнопка була клікнута.

3. Завдання TO-DO лист:

Створіть HTML-сторінку, яка містить список завдань (to-do list) з можливістю додавання нових завдань. Ваше ціль - використовуючи делегування подій, створити обробник подій для списку завдань, який дозволить видаляти завдання при кліку на них.
